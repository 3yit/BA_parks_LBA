# BA Parks LBA

A React + TypeScript application for exploring Bay Area parks.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

The app will start on `http://localhost:5173` (or another available port).
